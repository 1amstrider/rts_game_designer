# RTS Game Designer

Offline desktop application scaffold for creating, editing, and managing RTS hero and game data.

## Structure
- `assets/` for hero images and icons
- `database/` for SQLite storage
- `generated_excel/` for exported workbooks
- `modules/` for data and business logic
- `ui/` for desktop UI code
- `styles/` for QSS styling

## Run
python app.py
